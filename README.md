# 📚 Library Management System (LMS) - Backend

A robust backend for managing books in a library, built with **Express.js**, **TypeScript**, and **MongoDB**. It includes book management, schema validation, filtering, pagination, and CRUD operations.

---

## 🚀 Features

- 🔧 Built with Express.js & TypeScript
- 📦 MongoDB with Mongoose ODM
- ✅ Schema validation (Zod)
- 📚 Book CRUD operations
- 🔍 Filtering, searching, and pagination
- 🧠 Business logic layer
- 🧪 Static/Instance methods & Middleware
- ⚙️ Modular project structure
